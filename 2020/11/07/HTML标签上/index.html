2.HTML基本结构标签

2.1 第一个HTML网页

每个网页都有一个基本的结构标签（也称为骨架标签），页面内容也是在这些基本标签上书写。

HTML页面也成为HTML文档。

标签名 定义 说明
<html></html> HTML标签 页面中最大的标签，我们称为根标签
<head>&</head> 文档的头部 注意head标签中我们必须要设置的标签是title
<title><title> 文档的标题 让页面拥有属于自己的网页标题
<body></body> 文档的主体 元素包含文档的所有内容，页面内容基本都是放在body里面的

3.3.1文档类型声明标签

<!DOCTYPE>文档类型说明，作用是告诉浏览器用那种HTML版本来显示网页，必须写到第一行，不是HTML标签，只是声明标签。

这句代码的意识是：当前页面采取的是HTML5版本来显示网页的

3.3.2 lang语言种类

用来定义当前文档显示的语言。

  1. en定于语言为英语
  2. zh-CN定义语言为中文

简单来说就是说明该网页是英文网页还是中文网页，定义为英文等会出发谷歌浏览器的翻译功能。

这个属性对浏览器和搜索引擎还是有一定作用的。

3.3.3 字符集

字符集（Character Set）是多个字符的集合以便于计算机能够识别和存储各种文字。

在<head>标签内，可以通过<meta>标签的charset属性来规定HTML文档应该使用哪种字符编码。

charset常用的值有：GB2312、BIG5、GBK和UTF-8，其中UTF-8被称为万国码，基本包含了全世界所有国家需要用到的字符。

建议提倡使用UTF-8字符编码。

3.3.4 总结

  1. 以上代码都是VSCode自动生成的，基本不需要我们重写。
  2. <!DOCTYPE>文档类型说明标签，告诉浏览器采取HTML5版本来显示页面。
  3. <html lang=“en”>告诉浏览器或搜索引擎这是一个英文网站，本页面采取英文来显示。
  4. <meta charset=”UTF-8”>必须写的标签，采取UTF-8来保存文字，不写会显示乱码。

4. HTML常用标签

4.1 标签语义

标签语义就是就是标签的含义，即每个标签是用来干嘛的。

根据标签的语义，在合适的地方给一个最为合适的标签，可以让页面结构更加清晰。

4.2 标题标签（重要）

为了使网页具有语义化，通常会用到标题标签。HTML提供了6个等级的网页标签，即<h1>——<h6>。

单词head的缩写，意为头部、标题。

标签语义：作为标题使用，并且依据重要性递减。

特点:

  1. 加了标题的文字会变得加粗，字号也会依次变大。
  2. 一个标题独占一行。

4.3 段落标签和换行标签（重要）

在网页中，想要把页面有条理的显示出来，就需要将这些文字分段显示。在HTML标签中，<p>标签用于定义段落，它可以将整个网页分为若干个段落。

单词paragraph的缩写，意为段落。

标签语义：可以把HTML文档分割成若干段落。

特点：

  1. 文本在一个段落中会根据浏览器窗口大小自动换行。
  2. 段落和段落之间保佑空隙。

在HTML中，文字一般是到达服务器右端然后才自动换行。如果希望强制换行显示，需要使用换行标签<br />。

单词break的缩写，意为打断、换行。

标签语义：强制换行

特点：

  1. <br />是一个单标签。
  2. <br />标签只是简单的开始新的一行，跟段落标签不一样，段落之间会插入一些垂直的间距。

4.4 文本格式化标签

在网页中需要为文职设置出题，斜体或下划线等效果，这是需要用到HTML中的格式化标签，使得文字以特殊的方式显示。

标签语义：突出显示，比普通文字更重要。

语义 标签 说明
加粗 <strong></strong>或者<b></b> 更推荐使用<strong> 语义更强烈
倾斜 <em></em>或者<i></i> 更推荐使用<em> 语义更强烈
删除线 <del></del>或者<s></s> 更推荐使用<del> 语义更强烈
下划线 <ins></ins>或者<u></u> 更推荐使用<ins> 语义更强烈

先重点记住加粗和倾斜的标签。

4.5 div和span标签

<div>和<span>标签是没有语义的，它们就是一个盒子，用来装内容的。

div是division的缩写，表示分割、分区。span意为跨度、跨距。

特点：

  1. <div>标签用来布局，但是现在一行只能放一个<div>。大盒子
  2. <span>标签用来布局，一行上可以有多个span。小盒子

4.6 图像标签和路径（重点）

4.6.1 图像标签

  1. 在HTML中，<img>标签用于定义HTML页面中的图像。

  3. 单词image的缩写，意为图像。

  4. src是<img>标签的必须属性，它用于指定图像文件的路径和文件名。

  5. 所谓属性：就是属于这个图像标签的特性。

  6. 图像标签的其他属性：

  • 属性 属性值 说明
    src 图像路径 必须属性
    alt 文本 替换文本，图像不能显示的文字
    title 文本 提示文本，鼠标放在图像上的描述文字
    width 像素 设置图像的宽度
    height 像素 设置图像的高度
    border 像素 设置图像的边框粗细
  1. 图像标签的注意点：
  • 图像标签可以拥有多个属性，必须写在标签名的后面。
  • 属性之间不分前后顺序，标签名与属性、属性与属性之间均以空格分开。
  • 属性采用键值对的形式出现，即key=”value”的格式，属性=”属性值”。

4.6.2路径（前期铺垫知识）

  1. 目录文件夹和根目录
    • 实际开发中，文件不能随便乱放，否则很难快速找到。需要统的文件夹来管理他们。
    • 目录文件夹：就是普通文件夹，里面不过是存放了我们做页面所需要的相关素材，比如html文件、图片等。
    • 打开目录文件夹的第一层就是根目录。
  2. VSCode打开目录文件夹
    • 直接拖动文件夹
    • 新建==>打开文件夹

4.6.3路径

  1. 相对路径

    • 以引用文件作为产骄傲基础而建立出目录路径

    • 或者说图片相对于HTML页面的位置

    • 相对路径分类 符号 说明
      同一级路径 图像文件位于HTML文件的同一级 如<img src=”baidu.jpg” />
      下一级路径 / 图像文件位于HTML文件的下一级 如<img src=”/baidu.jpg” /&g
      上一级路径 ../ 图像文件位于HTML文件的上一级 如<img src=”../baidu.jpg” /&g

  2. 绝对路径

    • 是指目录下的绝对位置，通常是从盘符开始的路径。

4.7超链接表标签（重点）

在HTML标签中，<a>标签用于定义超链接，作用是从一个页面链接到另一个页面。

4.7.1 链接的语法格式

单词anchor的缩写，意为：锚。

两个属性的作用如下：

属性 作用
href 用于指定链接目标的URL地址（必须属性），当为标签应用href属性时，它就具有了超链接的功能
target 用于指定链接页面的打开方式，其中seif为默认值，blank为在新窗口中打开方式。

4.7.2链接的分类

  1. 外部链接，例如<a href=”http://www.baidu.com">百度</a>。
  2. 内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可，例如<a href=”index.html”>百度</a>。
  3. 空链接：如果当时没有确定链接目标时，<a href=”#”>空链接</a>。
  4. 下载链接：如果href里面地址是一个文件或者压缩包，会下载这个文件。
  5. 网页元素链接：网页中的各种网页元素，如文本、图像、表格、音频、视频等多可以添加超链接。
  6. 描点链接：点击某个链接，可以快速定位到页面中的某个位置。
    • 在链接文本的href属性中，设置属性值为**#名字**的形式，如<a href=”#two”>第二集</a>。
    • 找到目标位置标签，里面添加一个id属性=刚才的名字，如<h3 id=”two”>第二集介绍</h3>。

5.HTML中的注释和特殊字符

5.1 注释标签

网页中添加一些便于阅读理解的解释性语句担忧不需要显示在页面中的注释文字，就需要用到注释标签。

HTML中的注释以“<!–”开头，以“–>”结束。

简而言之，注释语句是给开发人员看的，这些注释代码不执行也不显示在页面中。

添加注释是为了更好的解释代码的功能，便于线管人员理解和阅读代码，程序是不会执行注释内容的。
